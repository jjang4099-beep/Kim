## 3. 구조 핵심 (자세한 건 CLAUDE.md)

```
1. 아카이브_mobile/public/
├── index_mobile.html     # 앱 진입점, JS/CSS는 ?v=N 캐시버스팅 (수정 시 버전 ↑ 필수)
├── js/  core → auth → home → feed/library/manage/add → app_exam → pwa
│   ├── core.js           # 전역 state·유틸·Mob={} 선언
│   ├── auth.js           # 로그인/회원가입/로그아웃·checkAuth (v87부터 추가)
│   ├── home.js           # 홈·공통카드·상세·모드전환(setMode)·검색·앱 진입(init)
│   ├── feed.js           # 지식 배달
│   ├── library.js        # 내 서재
│   ├── manage.js         # 통계·복습·퀴즈
│   ├── add.js            # 추가 모달
│   └── app_exam.js       # 수험생 모드 (ExamMob 네임스페이스)
├── css/ style_mobile.css (현재 v90) · exam.css (현재 v9)
├── db/                    # DB 접근 계층 — server.js 라우트는 여기 함수만 호출(raw SQL 직접 작성 금지)
│   ├── connection.js      # sql.js 커넥션 싱글턴
│   ├── items.js           # items CRUD — userId 인지 래퍼(getItemsByUser 등)가 기본 접근 경로
│   ├── users.js           # 인증 유저 CRUD (data/users.json 배달설정과는 별개 개념)
│   ├── categories.js      # user_categories CRUD
│   └── summaries.js       # 결산 캐시(userId 포함 캐시 키)
├── lib/
│   ├── auth.js            # JWT 발급/검증 + requireAuth 미들웨어
│   ├── domain.js           # 8대 도메인·모드 정규화 헬퍼
│   └── jsonStore.js        # 범용 JSON read/write
├── data/country_db/      # 여행 아카이브 국가 JSON (JP만 생성됨, npm run build-country로 확장)
├── data/
│   ├── archive.db        # SQLite(sql.js) 런타임 — git 미추적, 직접 편집 금지
│   ├── knowledge_db/     # 직장인 시드 (영어·중국어·역사)
│   ├── work_db/          # 직장인 신규 시드 (고전·고사성어·인사이트)
│   └── exam_db/          # 수험생 시드 (영단어·한국사)
└── server.js             # Express API + SQLite + 배달 생성 로직
```

**반드시 지킬 규칙 (위반 시 버그):**
- 전역 함수/변수 새로 만들지 말 것 → `Mob`/`ExamMob` 메서드, `state` 객체에만.
- `fetch()` 직접 호출 금지 → `fetchJSON()`.
- **모드 격리**: 모든 items 조회/저장에 `mode` 파라미터 필수. 직장인↔수험생 콘텐츠는 폴더·테이블까지 완전 분리.
- **유저 격리(07-12 추가)**: 모든 `/api/*`는 `requireAuth` 전역 가드 적용(예외는 server.js `PUBLIC_API_PATHS`). 유저 데이터는 `db/items.js`의 `getItemsByUser(userId, mode)` 등 userId 인지 함수만 사용 — `readDB()`/`dbInsert()` 등 무필터 함수는 배치 작업 전용.
- CSS 색상 하드코딩 금지 → CSS 변수(`--bg`, `--text-1`, `--border`, `--title-ink` 등).
- JS/CSS 수정하면 `index_mobile.html`의 `?v=N` 올릴 것.
